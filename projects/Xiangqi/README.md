## 🎮 项目：中国象棋 (Project: Chinese Chess)

*   **文件位置 (File Location)**：`projects/Xiangqi/Xiangqi_1.2.0.html`
*   **在线试玩 (Play Online)**：[点击这里](https://yang-2026-user.github.io/curly-parakeet/projects/Xiangqi/Xiangqi_1.2.0.html) （请确保文件名与仓库一致）

这是一个功能完整的**中国象棋**游戏，支持双人对战、人机对战，并内置了多级AI引擎。使用纯HTML、CSS和JavaScript编写。

This is a fully functional **Chinese Chess (Xiangqi)** game, supporting PVP and AI battles with a built-in multi-level engine. Written in pure HTML, CSS, and JavaScript.

### ✨ 主要功能 (Features)

*   **三种模式 (Three Modes)**：
    *   👥 **双人对战 (PVP)**：两人轮流在同一设备上对弈。
    *   🤖 **人机对战 (AI)**：可挑战“简单”、“普通”、“困难”、“大师”四档AI。
*   **交互与反馈 (Interaction & Feedback)**：
    *   🔊 **音效反馈 (Sound Effects)**：走子、吃子、将军、胜负均有提示音。
    *   🌐 **多语言 (Multi-language)**：支持中文和英文界面切换。
    *   ↩️ **悔棋 (Undo)**：支持悔棋（人机模式下AI会根据难度决定是否同意）。
*   **AI 难度 (AI Difficulties)**：
    *   `Easy` (简单)：基础搜索，适合新手。
    *   `Normal` (普通)：中等深度搜索，平衡挑战。
    *   `Hard` (困难)：深度搜索，开启开局库和杀手走法。
    *   `Master` (大师)：深度搜索，更强的计算能力。

### 🛠️ 本地运行 (Run Locally)
1.  下载 `projects/Xiangqi/Xiangqi_1.2.0.html` 文件。
2.  用浏览器（Chrome, Edge, Safari等）双击打开即可开始游戏。
