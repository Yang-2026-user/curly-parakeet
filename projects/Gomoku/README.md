##🎮 项目：五子棋游戏(项目:五子棋游戏)

*   **文件位置(文件位置)**：`projects/Gomoku/gomoku.html`
*   **在线试玩(在线播放)**：点击右侧“关于”区域的链接,或访问`https://yang-2026-user.github.io/curly-parakeet/projects/Gomoku/Gomoku_V1.0.1.html`（需先开启GitHub页面)

这是一个功能完整的**双人对战、人机对战**五子棋游戏,使用纯HTML、CSS和Java Script语言编写。

这是一个功能齐全的**PVP，AI**五子棋游戏，用纯HTML，CSS和JavaScript编写。

###✨ 主要功能(功能)

*   **三种模式(两种模式)**：
    *   👥 **双人对战(PVP)**：两人轮流在同一设备上对弈。
    *   🤖 **人机对战(大赦国际)**：可挑战"简单"、"普通"、"困难"三档艾.可选执黑或执白。
*   **交互与反馈(互动和反馈)**：
    *   🎨 **多款主题(主题)**：榧木、深蓝、翠绿、石墨，护眼又美观。
    *   🔊 **音效反馈(声音效果)**：落子、获胜、失败均有提示音。
    *   ↩️ **悔棋(撤消)**：支持悔棋（人机模式下人工智能会"思考"是否同意)。
*   **人工智能难度(AI难点)**：
    *   `容易的` (简单)：基础启发式评分，适合新手。
    *   `常态` (普通)：一步前瞻，平衡挑战性。
    *   `困难的`（困难):使用极小极大搜索(最小最大值)与阿尔法-贝塔剪枝,更具策略性。

###🛠️ 本地运行(本地运行)
1.  下载 `projects/Gomoku/gomoku.html` 文件。
2.  用浏览器（Chrome, Edge, Safari等）双击打开即可开始游戏。
